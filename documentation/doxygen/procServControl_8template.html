Detailed Description

This template allows monitoring of an IOC running through procServ in EPICS. It should be run in a separate IOC, but needs to be running on the same machine as procServ as procServ only allows localhost access

Gui snapshot

Parameters:
P Required argument. Device prefix
PORT Required argument. Asyn port name
SHOWOUT Default=1. Set this to 0 to disable the IOC output display
MANUALSTART Default=. Set this to # if the IOC is manually started. This changes the severity of STATUS
name Default=. Object and gui association name
